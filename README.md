# ChooseMeAGame
CMAG is a Python executable that picks a random game for you from a folder you provide with shortcuts of the games you want to include in CMAG.

## Note 
Program works with all types of shortcuts, so you can use it to randomly pick from any directory of shortcuts you want.

## How to Install
1. Download the executable provided in the [latest release](https://github.com/emreternity/ChooseMeAGame/releases)
2. The program will generate <ins>two more files</ins> where it's placed but <ins>will</ins> work no matter the location.
3. Collect all the shortcuts of your games in a folder and copy the <ins>absolute path</ins> of that folder.
4. Execute CMAG and when asked, provide the absolute path you just copied.
5. Use the menu to exclude games and let ChooseMeAGame ChooseYouAGame!
